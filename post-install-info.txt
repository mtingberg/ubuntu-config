

Post-install information and suggested actions
----------------------------------------------

* Git: complete an installation by configuring your git user name and email address:

$ git config --global user.name "Firstname Lastname"
$ git config --global user.email "firstname.lastname@example.com"


* Custom git prompt: in case of the following error:

bash: __git_ps1: command not found

See: https://gist.github.com/scottious/0d99ea77daa041b28929
